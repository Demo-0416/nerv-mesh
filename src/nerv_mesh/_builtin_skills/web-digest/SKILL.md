## Instructions

When asked to research or digest a topic:

1. Use `web_search` with 3-5 different query angles to find diverse sources
2. Use `web_fetch` to retrieve the top 3-5 most relevant pages
3. Cross-reference findings across sources
4. Produce a structured digest:

```
# Web Digest: [Topic]

## Overview
Brief introduction to the topic.

## Key Findings
- Finding 1 (source: ...)
- Finding 2 (source: ...)

## Different Perspectives
Note any conflicting or complementary viewpoints.

## Sources
1. [Title](URL)
2. [Title](URL)
```

Prioritize recent and authoritative sources. Flag any information that appears uncertain.

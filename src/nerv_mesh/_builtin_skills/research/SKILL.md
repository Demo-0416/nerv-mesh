## Instructions

When the user asks you to research a topic:

1. Break the topic into 3-5 focused search queries
2. Execute each search and collect relevant information
3. Cross-reference findings from multiple sources
4. Synthesize a structured report with:
   - Key findings
   - Supporting evidence
   - Conflicting viewpoints (if any)
   - Confidence assessment
5. Cite sources for all claims

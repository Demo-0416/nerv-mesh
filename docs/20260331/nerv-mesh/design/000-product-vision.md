# 000 产品愿景

## 项目名

- `nerv-mesh`

## 目标

构建一个面向 long-horizon 任务的智能体系统，核心差异化是：

- `sub-agents` 协作式任务分解与执行
- `多渠道接入`（优先飞书）
- `sandbox` 控制执行边界
- `tools` 统一接入层（skills、function calling、MCP）
- `memory` 创新系统：模拟人类睡眠/做梦，记忆以“突触网络”形式演化

## 非目标（当前阶段）

- 不追求一次性实现完整生产能力
- 不限定具体模型厂商或 API 提供方
- 不在本阶段绑定单一 UI 形态

## 成功标准（v1）

- 能持续处理跨天/跨轮次任务（long horizon）
- 子代理分工清晰，可观测可回放
- 具备可配置的安全沙箱
- 记忆系统能发生“巩固、衰减、联想、梦境生成”
